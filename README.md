# [sombochea.github.io](https://sombochea.github.io)

## Up and running

Install dependencies:

```bash
yarn
```

First, run the development server:

```bash
yarn dev
```

Build to production:

```bash
yarn build
```

Export to static files:
```bash
yarn export
```
